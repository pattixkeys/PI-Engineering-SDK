ClearBuffer() | Main Page

Overview

This function deletes all remaining data packets from the read buffer. This is useful if you want to void any user input that occured on the device before some program action (such as a warning message, or program start). If you call ReadLast() immediately after ClearBuffer(), there will be a slight delay before ReadLast() returns.
