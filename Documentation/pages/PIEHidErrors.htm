PIEHid32/PIEHid32Net Errors | Main Page

  • 0 = Success

Enumerate() errors

  • 101 = Bad HID device information set handle
  • 102 = No devices found in device information set
  • 103 = Other enumeration error
  • 104 = Error getting device interface detail (required size)
  • 105 = Error getting device interface detail
  • 106 = CreateFile error
  • 107 = HidD_GetAttributes error
  • 108 = VendorID not VID
  • 109 = HidD_GetPreparsedData error
  • 110 = HidP_GetCaps error
  • 111 = HidD_GetManufacturerString error
  • 112 = HidD_GetProductString error

SetupInterface() errors

  • 201 = Bad interface handle
  • 202 = Interface not valid
  • 203 = Already Connected
  • 205 = Cannot open read handle
  • 206 = Cannot open read handle - Access Denied
  • 207 = Cannot open read handle - bad DevicePath
  • 208 = Cannot open write handle
  • 209 = Cannot open write handle - Access Denied
  • 210 = Cannot open write handle - bad DevicePath

ReadData() errors

  • 301 = Interface handle (out of bounds)
  • 302 = Read length is zero
  • 303 = Interface not valid
  • 304 = Ring buffer empty
  • 305 = No read buffer
  • 307 = Handle Invalid or Device_Not_Found (probably device unplugged)
  • 308 = Handle Invalid or Device_Not_Found (probably device unplugged)
  • 309 = Read error (unplugged)
  • 310 = Bytes read not equal readSize
  • 311 = BlockingReadData timed out/dest.Length<ReportSize. This can occur if the user is trying to use BlockingReadData but did not turn off the callback first.

Write() errors

  • 401 = Bad interface handle (out of bounds)
  • 402 = Write length is zero
  • 403 = Bad internal interface handle/wData.Length<ReportSize
  • 404 = Write buffer full
  • 405 = No write buffer
  • 406 = Device not connected or bad CRC/Interface not valid
  • 407 = Unspecified write error/No writeBuffer
  • 408 = Bytes written not equal writeSize/Device Disconnected
  • 409 = Unknown write error
  • 410 = byteCount!=writeSize
  • 411 = Timed out in write
  • 412 = ReportID error

ReadLast() errors

  • 501 = Bad interface handle
  • 502 = Read length is zero
  • 503 = Interface not valid
  • 504 = Insufficient data (< readSize bytes)/No data yet
  • 507 = Handle Invalid or Device_Not_Found (probably device unplugged)

ClearBuffer() errors

  • 601 = Bad interface handle
  • 602 = Interface not valid

SetDataCallback() errors

  • 701 = Bad interface handle
  • 702 = Interface not valid
  • 703 = Could not create event thread/Input ReportSize zero
  • 704 = Only eventType changed/Data Handler Already Exists

SetErrorCallback() errors

  • 801 = Bad interface handle
  • 802 = Interface not valid
  • 803 = Could not create event thread
  • 804 = Error Handler Already Exists
