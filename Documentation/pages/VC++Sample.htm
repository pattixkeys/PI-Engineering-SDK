Visual C++ 2010 Sample | Main Page

Overview

The Visual C++ 2010 sample is included in the C:\Program Files\PI Engineering\PI Engineering SDK\Product\C++\PIHCallbackDemo.sln.

Executable located in C:\Program Files\PI Engineering\PI Engineering SDK\Product\C++\Release\PIHCallbackDemo.exe

A copy of PIEHid32.dll is required in either the project directory or the windows\system32 directory.

This sample makes use of the callback features of PIEHid32.dll. The Start button performs EnumeratePIE() and SetupInterfaceEx() calls which are necessary to establish communication. The None, New Data and Changed Data selection determine the type of callback desired using SetDataCallback(). None means no callbacks ever, New Data will send a callback when new data is received and Changed Data will callback only if the data changed from the last callback. The data is received in the HandleDataEvent function.
