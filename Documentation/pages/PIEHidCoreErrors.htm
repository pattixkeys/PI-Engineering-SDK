PIEHidCore Errors | Main Page

  • 0 = Success

SetupInterface() errors

  • 203 = Already Connected
  • 207 = Cannot open read handle, handle invalid
  • 208 = Cannot open write handle, handle invalid
  • 209 = Cannot open both read and write handles, both handles invalid

ReadData() errors

  • 302 = Read length is zero
  • 303 = Interface not valid
  • 304 = Ring buffer empty
  • 308 = Handle Invalid or Device_Not_Found
  • 309 = Read error (unplugged)
  • 310 = Bytes read not equal readSize
  • 311 = BlockingReadData timed out/dest.Length<ReportSize. This can occur if the user is trying to use BlockingReadData but did not turn off the callback first.
  • 320 = Cannot open read handle, handle invalid
  • 321 = Cannot open read handle, handle invalid

Write() errors

  • 402 = Write length is zero
  • 403 = wData.Length<ReportSize
  • 404 = Write buffer full
  • 405 = No write buffer
  • 406 = Interface not valid
  • 407 = No writeBuffer
  • 408 = Bytes written not equal writeSize/Device Disconnected
  • 409 = Unknown write error
  • 410 = byteCount!=writeSize
  • 411 = Timed out in write
  • 412 = ReportID error

ReadLast() errors

  • 502 = Read length is zero
  • 503 = Read length > data length
  • 504 = Insufficient data
  • 507 = Interface not valid

SetDataCallback() errors

  • 702 = Interface not valid
  • 703 = Input ReportSize zero
  • 704 = Data Handler Already Exists

SetErrorCallback() errors

  • 802 = Interface not valid
  • 804 = Error Handler Already Exists
