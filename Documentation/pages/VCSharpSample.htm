Visual C# 2008/2010 Sample | Main Page

Overview

The Visual C# 2008/2010 sample is included in the C:\Program Files\PI Engineering\PI Engineering SDK\Product\C#\PIEHidDotNet Sample.sln

Executable located in C:\Program Files\PI Engineering\ PI Engineering SDK \Product\C#\PIEHidDotNet Sample\bin\PIEHidDotNet Sample.exe

A copy of PIEHid32Net.dll is required in either the project directory or the windows\system32 directory.

This sample makes use of the .NET class library PIEHid32Net. This sample already has PIEHid32Net added as a reference but if starting with a new project one must right click on the project name (PIEHidDotNet Sample) and select Add Reference. Click on the Browse tab and select PIEHid32Net.dll. Also add the line 'using PIEHid32Net;' to the top of the code.
