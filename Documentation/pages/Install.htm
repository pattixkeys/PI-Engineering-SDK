Installing/Uninstalling P.I. Engineering SDK | Main Page

To Install:

  1. Exit any applications you are running.
  2. Get the installer PIEngineeringSDK.exe either from the P.I. Engineering Developerâ€™s CD or by downloading it from www.xkeys.com. Run PIEngineeringSDK.exe.
  3. Read the Welcome screen, then click Next to install P.I. Engineering SDK
  4. Read the Software License Agreement, and if you agree to the terms, click Yes.
  5. Follow the installation instructions.

To uninstall:

  1. Select Add/Remove Programs from the Control Panel.
  2. Locate P.I. Engineering SDK* in the list and select it.
  3. Click Remove and follow any instructions when prompted.
